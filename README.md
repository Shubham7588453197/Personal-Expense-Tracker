Personal Expense Tracker
A lightweight, open-source tool designed to help you track daily spending, monitor budgets, and gain financial clarity through simple, intuitive visualizations.

Features
Fast Logging: Record transactions in seconds.

Visual Insights: Understand where your money goes with clear, interactive charts.

Budgeting: Set and track custom limits for different categories.

Privacy-Focused: Your financial data stays under your control.

Getting Started
Clone the repo: git clone [your-repo-url]

Install dependencies: npm install

Run locally: npm start
